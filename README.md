# SwiftPaper

**SwiftPaper** 能够快速地查询 CCF 计算机推荐期刊、会议列表，包含查询该会议、期刊的详细信息，访问会议、期刊的网址以及会议征稿信息等功能。

期刊、会议的简称、全称是什么？领域是什么？出版社是哪一家？dblp网址是什么？在这里您都可以查到。

会议什么时候截稿？会议举办地是哪里？增加全新查看会议征稿信息功能！截稿倒计时告诉您还有多久截稿！
